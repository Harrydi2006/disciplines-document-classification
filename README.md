# 文件分类助手

基于人工智能的文档分类工具，能够自动识别并分类不同学科的文档。

## 功能特点

- 文本文件分类
- OCR 图片识别
- 音频文件转写
- 压缩包处理
- 多线程并行处理
- 实时日志显示

## 环境要求

- Python 3.8+
- Tesseract OCR
- FFmpeg
